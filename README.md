# React SCADA HMI

![SCADA HMI Interface](https://via.placeholder.com/800x400.png?text=SCADA+HMI+Interface)

A modern web-based SCADA/HMI system built with Next.js, React Flow, and MQTT for real-time industrial process monitoring and control.

## Features ‚ú®

- **Real-time Communication**  
  MQTT integration for live data updates
- **Interactive Diagrams**  
  React Flow-based process visualization
- **25+ Industrial Components**  
  Valves, Motors, Tanks, Pumps, Gauges
- **Custom Dashboards**  
  Drag-and-drop interface builder
- **Data Visualization**  
  Trend charts with D3.js and ApexCharts
- **Alarm System**  
  Priority-based alerts with visual indicators
- **Cloud Ready**  
  Terraform AWS deployment + CI/CD pipeline
